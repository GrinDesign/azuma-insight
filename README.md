# 名言集アプリ（azuma-insight）

## 概要
SupabaseとFastAPIを使った名言集アプリ。名言の登録・検索・感想記録ができます。

## 目的
- 名言をデータベース化し、条件に合った名言を引き出す
- 感想を記録し、分析も行う

## 開発環境
- Python 3.x
- FastAPI
- Supabase

## セットアップ手順
1. 仮想環境の作成
2. 必要なライブラリのインストール
3. Supabaseプロジェクトの準備
